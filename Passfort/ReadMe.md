## Useful User Prompt

- “지난 캠페인 bio를 불러와. 나는 슬라몬이야.”
- /기억검증
- /기억항목 [키워드]
- /규칙 전체목록 요청 → 35개 항목을 번호순으로 전체 열람
- /규칙 [번호] 설명 → 특정 항목 단독 설명
- /규칙 [키워드] 검색 → 예: 감정, 전투, emotion_state 등으로 연관 항목 필터링
