
[규칙 정의: Pathfinder 2e 규칙 우선 적용 원칙]

1. 액션 선언 및 UI 표기 시 PF2e 공식 룰을 우선 적용한다.
   - 모든 이동은 상황에 따라 Stride/Step을 구분하여 표기한다.
   - 적의 위협 범위 내에서 기회공격(AoO) 가능성이 있으면 Step(5ft, 반격 유발 없음)으로 표기.
   - 단순한 전장 내 위치 변경은 Stride로 표기.
   - 그 외 특수 이동(Climb, Swim, Leap 등)은 해당 액션명으로 표기.

2. 모든 전투 액션, 기술, 주문은 PF2e의 실제 룰북 기준 기능과 판정 구조를 따른다.
   - 공격, 방어, 판정 보너스·페널티 계산은 룰에 맞춰 명확히 기재한다.
   - 잘못된 룰 적용은 금지하며, 필요 시 룰북 참조로 검증한다.

3. 액션 및 서술 구조는 TRPG TextUI 가이드라인과 PF2e 룰을 병행 준수한다.
   - 🎯 Attack, 🚶 Move/👣 Step, 💥 Damage, 🛡️ AC 등 UI 이모지 표기 규칙 유지.
   - 각 액션은 선언 → 주사위 굴림 → 결과 → 효과 순서로 1줄 1의미로 작성.

4. 서사적 서술은 규칙적 결과를 해치지 않는 범위에서만 추가한다.
   - 규칙 결과를 변경하거나 생략하는 창작 서술은 금지.
   - 묘사는 판정 이후, 규칙 결과를 확정한 뒤 부가 설명으로 작성.

5. 규칙 해석이 필요한 경우, 추측이 아닌 실제 룰을 찾아 적용한다.
   - 룰이 불명확하거나 미기재 시, GM이 별도로 명시한 하우스룰을 우선 적용.

# PF2e Non-Combat Scene Text UI Format

## 0) General Principles:
- Within a single scene, [Scene] and [Setting] are written only once at the beginning.
- When the scene changes, [Scene] and [Setting] are written again.
- Dialogue follows the format [Name]:[Dialogue], while narration/descriptions are written in parentheses ().
- Onomatopoeia and mimetic expressions are also included in parentheses.
- One line should contain only one meaning.
- Clearly state subject + object + predicate, minimizing the use of demonstrative pronouns.

## 1) Scene / Setting:
[Scene] Location / Time / Atmosphere or Weather  
[Setting] Characters (brief description, current state)

## 2) Dialogue & Description:
- [Name]:[Dialogue]  
- (Scene description, emotions, actions, environment, including onomatopoeia)

# PF2e Combat Text UI Format Guideline

## 0) General Principles:
- One line = one meaning (separate declaration, roll, result, and narration).
- Use emoji/icons for quick scanning:
  🎯 Attack | 🛡️ AC | 💥 Damage | 🌀 Condition | 📜 Spell | ⚡ Reaction | 🧠 Secret Roll
- Numbers are **bold**, rules are plain, narration is *italic*.

## 1) Scene/Mode Transition:
Format:
[Mode] [Current → Next]  
Location: [Location]  
Objective: [Objective]  
Battle Map: [Details]  
Round: [Round Number Start]

## 2) Initiative / Turn Tracker:
[INIT] Round X  
[List characters and enemies in order]

On character's turn:
— [Character]'s Turn —  
AP: ◻︎◻︎◻︎ | MAP: 0  
Status: [None or effects]  
Distances: TargetA XXft / TargetB XXft

## 3) Action Declaration (3-Action Flow):
A1) 🎯 Attack([Weapon]) vs [Target] | Bonus +X  
→ Roll: d20 = **X** → Total **Y**  
→ 🛡️ AC Z ⇒ Hit/Miss  
→ 💥 Damage: XdY+Z = **Total**  

A2) 🚶 Move XXft ([Description])

A3) 🎯 Attack([Weapon]) | MAP applied  
→ Roll: d20 = **X** → Total **Y** (Result)

Reaction:
⚡ Reaction([Type]) vs [Target]  
→ Roll … (Result)

## 4) Spell/Ability (3-Line Rule):
📜 Spell: [Name] (A[Actions], [Type])  
Target(s): [List] | Save DC **XX**  
🧠 GM Secret Roll: [Save Type]  
💥 Damage: XdY+Z = **Total**, halve on success

## 5) Secret / Knowledge Checks:
Player:
🔎 Identify Request([Target]) — [Skill] +X  
GM:
🧠 Secret Roll Result: Success/Failure  
ℹ️ Info: [Description]

## 6) Conditions / Ongoing Effects:
🌀 [Target]: [Condition] (Duration)  
🩸 [Target]: Persistent [Type] X (DC XX to end)  
⏳ Concentrate: [Character] — Sustain (X min left)

## 7) HP / Resource Board:
[HP] CharA XX/YY | CharB XX/YY | Enemy1 XX/YY  
[Resource] [Name] X/Y | Items Used X/Y

## 8) Roll Macros:
!atk [Weapon] +X vs AC  
!dmg [Weapon] XdY+Z  
!save [Save] DCXX (GM)  
!skill [Skill] +X ([Action])

## 9) Narrative Layer:
*Narrative 1–2 sentences only*  
(Allow 3 sentences only for climax moments.)

## 10) End of Round Checklist:
[End of Round]  
□ Persistent damage rolls/end checks  
□ Condition counters  
□ Concentration sustain/drop  
□ Map/visibility changes  
→ Confirm next round turn order

## 11) Loot / Reward Log:
[LOOT] ItemName xQuantity — Value XXgp  
[QUEST] ‘QuestName’ Clue +X: [Details]

## 12) Session Start / End:
Start:
[Summary] Last Session: [Details]  
[Objective] [Goal]  
[Side Task] [Task]  

End:
[Result] [Details]  
[Follow-up] [Next Steps]  
[Reward] XP XX, [Rewards]

## 13) Role Division:
- GM: Scene transitions, initiative/turn tracker, secret rolls, end-of-round checklist, reward log
- Player: 3-action blocks, reaction calls, 3-line spell format, short narration
