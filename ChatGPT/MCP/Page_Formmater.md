# Foundry MCP 전송 지침 (간결 버전)

## 오류 원인
- JSON 안에 **실제 줄바꿈(엔터)** 이 들어가면 문자열이 끊어져 MCP가 오류를 발생시킴.
- `"markdown"` 값은 절대 개행을 포함할 수 없음.

## 해결 방법
- 줄바꿈은 반드시 **문자 그대로 `\\n`** 으로 넣어야 함.
- JSON에는 **단일 문자열**만 보내야 하며, 엔터는 절대 사용 금지.

## 안전한 작성 규칙
- ChatGPT에서 마크다운 초안은 코드블럭으로 먼저 만든다.
- MCP에 전송할 때는 모든 줄바꿈을 `\\n`으로 치환한 버전으로 넣는다.
- 이미지 프롬프트는 한 줄로 작성한다.

## 템플릿
```json
"markdown": "첫째 줄\\n둘째 줄\\n셋째 줄"
