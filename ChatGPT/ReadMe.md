# ChatGPT를 사용한 TRPG

## json 구조의 정확성을 보장

- 프로젝트 파일은 초기에 .json 파일은 단순한 텍스트 포맷으로 인식
  - json.load() 전에는 구조의 정확성은 절대 보장할 수 없다.
  - 파일명, 이전 경험, 포맷 힌트 등으로 구조를 추정할 수는 있지만 보장할 수는 없음
- json.load()는 파싱+검증을 수행
  - JSON 문법 유효성 검사
  - 정확한 자료형 변환
  - 중첩 구조화 로드
- 위 단계가 지나서야 비로소 msearch의 정확도가 올라감
