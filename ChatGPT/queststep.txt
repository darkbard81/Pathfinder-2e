1. 도입 → 현상 인지 / 의뢰 접수
   └─ 플레이어가 '이상한 무언가'에 노출되는 순간
   └─ 감각의 틈을 통해 일상에서 사건으로 끌려들어감
   └─ 연출 키워드:
        • 기시감, 불쾌한 정적, 반복되는 꿈
        • NPC의 동요, 기록의 누락, 무형의 부름
   └─ 시스템 활용:
        • DC 없는 감각 묘사
        • 기본 Sense Motive / Occultism 판정 가능

2. 탐색 → 환경 분석 / 단서 추적
   └─ 플레이어가 대상 장소・사건을 조사하고 진입
   └─ 감각, 공간, 구조 자체가 단서를 품고 있음
   └─ 연출 키워드:
        • 반사된 감정, 익숙한 냄새, 살갗의 소름
        • 공간이 살아있는 듯한 반응
   └─ 시스템 활용:
        • Perception / Stealth / Lore / Occultism
        • 감각 중심의 Hazard 트리거 가능

3. 충돌 → 판정 발생 / 이벤트 발동
   └─ 접촉이 실체화되고, 감각이 플레이어를 덮침
   └─ 전투 또는 절정 Save 판정 발생
   └─ 연출 키워드:
        • 촉수의 압박, 혀 없는 감촉, 안쪽으로 미끄러지는 감정
        • 의도치 않은 반응, 육체의 배신
   └─ 시스템 활용:
        • Will / Fort Save DC 22~28
        • 상태 효과 (Orgasm, Stupefied, Flat-footed 등)
        • 전투 겸용 인카운터 가능

4. 분기 → 선택지 부여 / 역할 분기
   └─ 감응/기억/감정의 동기화 또는 거부 발생
   └─ 플레이어가 수용/저항/재정의 중 하나 선택
   └─ 연출 키워드:
        • 기억의 조작, 감정의 환기, 이성과 쾌락의 균열
        • 거짓된 진실, 자아의 감각 분열
   └─ 시스템 활용:
        • Diplomacy / Performance / Will Save
        • 선택지에 따른 서사・NPC・보상 경로 차별화

5. 해결 → 보상 정리 / 후속 연결
   └─ 감각적 사건은 종료되지만 흔적은 남음
   └─ 보상, 후유증, 관계 변화, 능력 변동 처리
   └─ 연출 키워드:
        • 정화되지 않은 쾌락의 잔재
        • 피부 아래 남은 감촉, 언어로 설명할 수 없는 기억
   └─ 시스템 활용:
        • Feat/Item 획득 or Corruption 부여
        • Persistent Save (1/day, Will DC 20) 생성 가능
        • 감정 상태 변화, 연속 시나리오 트리거 가능
